## 암호화

* 사용자가 서비스에 비밀번호를 맡겨놓은 상태. 

* 사용자의 비밀번호가 악의적인 사용자에게 넘어갈 경우를 대비
* 사용자의 비밀번호가 유출되면 안됨.
